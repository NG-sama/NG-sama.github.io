## Overview

This is a GitHub Pages personal site repository (`NG-sama.github.io`) — a single-page retro/pixel-art landing page linking out to the owner's other properties (Substack, LinkedIn, Medium, GitHub, Luma). Plain HTML/CSS/JS, no framework, no build system.

## Getting Started

No build tooling or dependencies — just `index.html`, `styles.css`, `script.js`, and `assets/avatar.png`. Serve locally with `python3 -m http.server 8000` and visit `http://localhost:8000/`. GitHub Pages serves `index.html` directly from the repo root.
